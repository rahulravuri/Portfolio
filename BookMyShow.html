Book My Show

This is a web-based movie ticket booking system that allows users to view movie showtimes listings and booking tickets for selected seats.

About The Project

This project is designed as a simulation of an online ticket booking system similar to BookMyShow. It’s built using modern web technologies and aims to provide users with an intuitive, easy-to-use interface for booking movie tickets online.

Key Features:

  • Implemented signup and login functionality in the BookUmShow application, enabling secure user authentication and seamless access to the platform.
  • Browse Movies and view available showtimes and theaters effortlessly.
  • Book Tickets, select preferred seats and complete bookings online with ease.
  • Update personal details and access past booking history conveniently

With a user-friendly interface and secure payment integration, this application aims to replicate the core features of BookMyShow.

Tools Used

  • Backend:Java SpringBoot
  • Database: MySQL
  • Authentication: JWT, OAuth
  • Deployment: Docker

Microservices Overview

Auth Service

Handles authentication and JWT-based authorization.

Gateway Service

Routes requests to correct services efficiently.

Movie Service

Manages shows and Booking creation.

Payment Service

Processes payments for Movie Service.

Database

Stores data with indexing for quick retrieval.
