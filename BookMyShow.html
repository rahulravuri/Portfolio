Book My Show

This is a web-based movie ticket booking system. The application allows users to view movie listings, select showtimes, choose seats, and book tickets.

About The Project

This project is designed as a simulation of an online ticket booking system similar to BookMyShow. It’s built using modern web technologies and aims to provide users with an intuitive, easy-to-use interface for booking movie tickets online.

Key Features:

  • Browse movie listings and filter by genre, language, etc.
  • View available showtimes and theaters in their area.
  • Choose seats for a show and book tickets online.
  • Manage their profile and view past bookings.

With a user-friendly interface and secure payment integration, this application aims to replicate the core features of BookMyShow.

Tools Used

  • Frontend: HTML, CSS, JavaScript, React
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: JWT, OAuth
  • Deployment: AWS, Heroku
